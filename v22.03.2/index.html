libsa API documentation#

Usage#

To use libsa from C/C++ just include ‘<sa.h>’ in your project.

  • Index

API#

SA_BOOL#

SA_BOOL booleans are non-zero integers for true, zero for false.

type sa_alloc_t#

A memory allocation function that can be used by this implementation.

Param size

(in) A size in bytes to allocate.

type sa_free_t#

A memory freeing function that can be used by this implementation.

Param p

(in) A pointer to memory to be freed.

sa_mm_fns_align#

The alignment of the memory management functions structure.

struct sa_mm_fns#

A matched pair of memory allocation and free functions.
